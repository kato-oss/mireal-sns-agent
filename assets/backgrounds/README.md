# assets/backgrounds/

加藤さんが用意するブランド画像の格納場所。

## 想定2モード

### モードA: 完成画像をそのまま投稿
- ファイル名: 任意（例: `2026-05-28.jpg`, `case-retail.jpg`）
- サイズ: **1080 × 1080 px (正方形)** 推奨（IG feed最適）
- 形式: **JPG** または **PNG**
- ファイルサイズ: 8MB以下
- 中身: テキスト含む完成デザイン

エージェントは画像を**そのまま**投稿し、テキスト合成はしない。

### モードB: 背景テンプレートとして使用（推奨）
- ファイル名: `bg-01.jpg`, `bg-02.jpg` ... と通し番号
- サイズ: **1080 × 1080 px**
- 中身: **テキスト無しの背景**（ロゴ・装飾・余白のみ）
- テキストが置かれる予定の中央エリア（縦400〜800px / 横100〜980px）は**素の背景色**で空けておくこと

エージェントは PIL で heading/subheading/footer を**背景の上に合成**して投稿。
1枚あたり数十日使い回せる。

## 命名規則

- モードA: 自由（agent はファイル名でソートして使用順を決める）
- モードB: `bg-XX.jpg`（数字でローテーション）

## 配置例

```
assets/backgrounds/
├── bg-01.jpg     # 抽象幾何
├── bg-02.jpg     # グラデーション
├── bg-03.jpg     # ロゴ装飾
├── bg-04.jpg     # シンボルアート
└── bg-05.jpg     # ミニマル
```

## 使われ方の制御

`scripts/agent.py` 内で `IMAGE_MODE` を `"user_as_is"` / `"user_bg_overlay"` / `"pil_only"` のいずれかに設定。

デフォルトは `"pil_only"`（このフォルダが空なら自動で PIL生成にフォールバック）。
